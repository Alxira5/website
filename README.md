# My personal website

![Screenshot](/static/img/blog/docusaurus-the-engine-of-my-website/main-page.png)

In this repository you will find the source code of my website, which works with [Docusaurus v3](https://docusaurus.io/).

You can see it in action at [alxira5.vercel.app](https://alxira5.vercel.app).

## Development

If you want to run my website on your computer, you must meet the following requirements:

* Node.js v18+
* npm v10+

Once you meet the requirements, you must execute the following commands:

```bash
git clone https://github.com/Alxira5/website.git
cd website
npm install
npm start
```

And ready, you are already running my website.
